## Why Accessibility Matters

Over 1 billion people worldwide have some form of disability. Building accessible applications isn't just ethical — it's good for business and required by law in many jurisdictions.

## Semantic HTML

Always start with semantic elements. A `<button>` is better than a `<div onClick>`.

## Keyboard Navigation

Ensure all interactive elements are reachable and operable via keyboard. Use `tabIndex`, `onKeyDown`, and focus management.

## ARIA Attributes

Use ARIA attributes sparingly and correctly. First rule: if you can use a native HTML element that has the semantics you need, don't use ARIA.

```tsx
function Button({ children, ...props }) {
  return (
    <button
      className="rounded-lg px-4 py-2"
      {...props}
    >
      {children}
    </button>
  );
}
```

## Testing with Playwright

Use Playwright's accessibility snapshot feature to catch regressions automatically.
